# Duplicates — Контекст

Еженедельный отчёт: дубли подписчиков за прошлую неделю.

## Файлы

| Файл | Назначение |
|------|-----------|
| `duplicates.sql` | Запрос: подписки с признаком дубля за прошлую неделю. Дата считается автоматически через `date_trunc('week', current_date)`. |
| `data/` | Исторические выгрузки по неделям (с 2025-12-22). |

## Источники данных

- `data_bronze_supabase_prod.smartyme_public__subscriptions` — подписки SmartyMe (Adapty)

## Как обновлять

Запрос не требует изменения дат — берёт прошлую неделю автоматически.
Новые выгрузки сохранять в `data/` с именем `duplicates_from_<start>_to_<end>.csv`.
