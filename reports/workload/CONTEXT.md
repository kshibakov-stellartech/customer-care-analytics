# Workload Modeling — Контекст

Моделирование нагрузки на агентов поддержки.

## Файлы

| Файл | Назначение |
|------|-----------|
| `workload_modeling.sql` | Модель нагрузки: тикеты и события по агентам с учётом их schedule. Содержит словарь агентов (`agent_dict_id`) с именами и Zendesk ID. |

## Источники данных

- `data_silver_zendesk_prod.zendesk_events`
- Agent IDs захардкожены в `agent_dict_id` CTE внутри запроса

## Примечание

При добавлении/уходе агентов — обновить словарь `agent_dict_id` в запросе.
