# User Scopes — Контекст

Еженедельный отчёт: скоупы (features) пользователей за прошлую неделю.

## Файлы

| Файл | Назначение |
|------|-----------|
| `users_scopes.sql` | Запрос: скоупы подписок за прошлую неделю. Дата считается автоматически. |
| `data/` | Исторические выгрузки по неделям (с 2025-12-22). |

## Источники данных

- `data_bronze_supabase_prod.smartyme_public__subscriptions` — подписки SmartyMe (Adapty)

## Как обновлять

Запрос не требует изменения дат.
Новые выгрузки сохранять в `data/` с именем `user_scopes_from_<start>_to_<end>.csv`.
